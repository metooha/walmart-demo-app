# Changelog

All notable changes to the Living Design 3.5 Portable Kit will be documented in this file.

Format: [version] - date

---

## [1.0.0] - 2025-06-01

### Components
- 50+ pre-built components: Button, TextField, Card, Modal, Tabs, Tag, OLQTag, Select, Popover, Panel, and more
- All components use semantic design tokens — no hard-coded values
- Full interactive state coverage: hover, focus, active, disabled
- WCAG 2.1 AA accessibility built into every component
- Component Sandbox for interactive property testing

### Design Tokens
- 648 semantic tokens (colors, typography, spacing, elevation, borders)
- 364 primitive tokens
- All values resolve through CSS custom properties

### Themes
- Walmart theme (default)
- Sam's Club theme
- Theme switcher component for runtime brand switching

### Icons
- 303 icons in the standard library (20x20 viewBox, currentColor fill)
- Custom icon slot (icons-custom/) for project-specific icons

### Internationalization
- English, Spanish, and French translations
- All UI strings use i18n translation keys

### Template Pages
- Home, Catalog, Landing Summary, Landing Connection, Detail Item, Page Template
- These are optional — remove them if not using Partner Experiences

### Documentation
- Getting Started guides for Designers, Developers, and AI Agents
- Component guidelines in guidelines/ folder
- Enforcement rules in guidelines/rules/
- Agent integration instructions (existing project, version upgrade, Figma-to-Code)

### Tooling
- Component Library layout with sidebar navigation
- Guidelines doc index
